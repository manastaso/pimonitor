# PiMonitor

A WeatherStation with
- JavaFX
- Akka (https://www.akka.io/)
- OpenMeteo (https://open-meteo.com/)
- Greek Namedays (https://www.eortologio.net/)
- and with the help of tilesfx (https://github.com/HanSolo/tilesfx)
